Work duration ------------- Add the following text at the end of the URL and replace 25 with your choice of minutes - &work=25 Break duration ------------- Add the following text at the end of the URL and replace 5 with your choice of minutes - &break=5 Auto-start ---------- If you want the work timer to start automatically, then add the following text - &autostart=true Auto-break ---------- If you want the break timer to start automatically, then add the following text - &autobreak=true Keep in mind that the value of &autobreak is set to that of &autostart by default. So, if you have set &autostart=true, it automatically implies &autobreak=true. In that case, if you want to disable auto-break, you need the set the following - &autobreak=false NOTE: The case of "true" and "false" is important. It should match exactly. Anything else is ignored. Help on Enter ------------------ By default, help is toggled when Enter is pressed. To disable this, add the following to the URL - &enterhelp=false
